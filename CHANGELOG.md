# Change Log

## [0.2.0] - 2026-02-22
- Added `watchers` array to support multiple file watchers
- Added `autoSave` action to save changed files to VS Code Local History
- Added `triggerOnVSCodeSave` option (default: true) to control reaction to VS Code saves
- Added `triggerOnExternalSave` option (default: true) to control reaction to external program saves
- Added `openLabel` option for the notify action button label
- Added English comments throughout the source code
